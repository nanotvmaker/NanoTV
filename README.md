## ✨ Features / 主要功能

### 🚀 Core Experience / 核心体验
- **Hardware Acceleration**: Supports H.264/H.265 encoding for smooth HD playback.
- **Multi-Platform**: Seamlessly sync subscriptions and settings via **iCloud** across iOS and tvOS.
- **Modern UI**: Full support for Light, Dark, and System modes.
- **硬解播放**: 支持 H.264/H.265 硬件加速，高清流畅。
- **多端同步**: 支持 iCloud 同步，在 iPhone, iPad 和 Apple TV 间无缝切换。
- **外观模式**: 完美支持深色、浅色及跟随系统模式。

### 📺 Playback & Sources / 播放与源管理
- **Flexible Subscriptions**: Supports **M3U**, **TXT**, **Xtream Codes API**, and **TVHeadend**.
- **Remote Management**: Manage playlists remotely via phone or computer.
- **EPG Support**: Electronic Program Guide with automatic translation (English ↔ Chinese).
- **多格式支持**: 支持 M3U/TXT, Xtream Codes, TVHeadend 等多种订阅方式。
- **远程管理**: 手机/电脑远程管理播放列表。
- **EPG 指南**: 支持节目单显示及自动翻译。

### 🤖 AI & Accessibility / 智能辅助
- **Real-time Subtitles**: AI-driven speech-to-text (English, Spanish, German, French, Italian).
- **Live Translation**: Real-time translation of subtitles.
- **实时字幕**: 支持多语种实时语音转文字。
- **字幕翻译**: 提供实时的多语言字幕翻译功能。
